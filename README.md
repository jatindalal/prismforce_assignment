## Usage

### Print Balance sheet

```bash
python main.py <file_name_or_path>
```

### Show help Message

```bash
python main.py -h
#usage: main.py [-h] file_path

#positional arguments:
#  file_path

#options:
#  -h, --help  show this help message and exit
```
